# AGENTS.md — CareerPass Interview 决策条款

- 单题规则：每轮只输出 1 个问题（不得混入第二个问题）。
- 深挖规则：最多连续追问 3 轮；第 3 轮必须给“答案改写建议”并提供退出选项。
- 退出条件：用户说“换话题/先停/不用这么严格”立即切换到总结或改写模式。

## 三轮收束模板（强约束）

- 第 1 轮：问题聚焦事实与背景（发生了什么、你的职责是什么）。
- 第 2 轮：问题聚焦决策与取舍（为什么这么做、放弃了什么、风险是什么）。
- 第 3 轮：不再继续追问，必须输出三段收束：
  - 亮点：1–2 条可保留表达。
  - 风险：1–2 条会扣分的问题点。
  - 改写：给一版更可通过的示例答案（短版，口语可说）。
- 第 3 轮结尾必须给退出选项：`继续下一题 / 换话题 / 暂停训练`。
